### transform_draft_csv
If you wish to run a certain year, check BBB_SEASON in .env

Run in the command line: `python ./scripts/transform_draft_csv.py`

data will output to data/config/draft_picks.csv

### build_playoff_game_ids.R
If you wish to run a certain year, check BBB_SEASON in .env

Run in the command line: `Rscript build_playoff_game_ids.R`

data will output to data/config/playoff_game_ids_{BBB_SEASON}.csv
